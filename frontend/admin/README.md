## Run Locally

Clone the project

```bash
  git clone https://github.com/Kevin310703/HotelBookingManagement-Website.git
```

Go to the project directory

```bash
  cd frontend/admin
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
